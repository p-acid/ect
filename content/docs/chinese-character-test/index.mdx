### 寺(절 사)

| 한자 | 음  | 뜻        | 구성 성분       | 설명                        |
| ---- | --- | --------- | --------------- | --------------------------- |
| 寺   | 사  | 절, 절 사 | 자체            | 절이나 관청 등을 의미       |
| 時   | 시  | 시간, 때  | 日(해) + 寺(절) | 해가 절에 있음 → 때, 시간   |
| 詩   | 시  | 시(시문)  | 言(말) + 寺(절) | 절에서 말함 → 시문, 문학    |
| 持   | 지  | 가지다    | 扌(손) + 寺(절) | 손으로 절처럼 공손히 가지다 |
| 特   | 특  | 특별하다  | 牛(소) + 寺(절) | 절에 바치는 특별한 소       |

| 經   | 경  | 경전, 지나가다  | 糸(실) + 巠(지날 경)             | 실처럼 이어진 글 → 경전, 혹은 줄기   |

| 綿   | 면  | 솜, 부드러운 실 | 糸(실) + 帛(비단 백)             | 실을 풀어 만든 솜                    |

## 비슷한 한자

| 한자 (비교)  | 단어 예시 1   | 단어 예시 2            | 설명                                    |
| ------------ | ------------- | ---------------------- | --------------------------------------- |
| 字 / 子      | 글자(字)      | 아들(子)               | 글자 자(字)를 아들 자(子)로 바꿔서 출제 |
| 要 / 腰      | 중요할 요(要) | 허리 요(腰)            | 要를 腰로 바꿔 출제, 요통(腰痛)         |
| 工 / 攻      | 공부(工夫)    | 전공(專攻)             | 工은 장인, 攻은 공격·전공 의미          |
| 競 / 兢      | 경주(競走)    | 전전긍긍(戰戰兢兢)     | 競은 립(立), 兢은 십(十) 포함           |
| 提 / 是      | 提 (끌다)     | 是 (바르다)            | 提는 손수변(扌), 是는 손 없음           |
| 墮 / 墜      | 타락(墮落)    | 추락(墜落)             | 모양이 비슷해 자주 혼동                 |
| 買 / 賣      | 사다(買)      | 팔다(賣)               | 賣 위엔 선비 사(士), 선비는 가난함      |
| 希 / 稀      | 희망(希望)    | 희소(稀少), 희미(稀微) | 稀는 벼 화(禾) 포함                     |
| 態 / 能      | 상태(狀態)    | 능력(能力)             | 熊은 能에 불 화(灬) 추가                |
| 品 / 器      | 물건(品)      | 그릇(器)               | 器는 개(犬)가 포함됨                    |
| 是 / 提      | 바르다(是)    | 들다(提)               | 提는 손수변(扌)이 있음                  |
| 直 / 値      | 곧다(直)      | 가치(値)               | 곧은 사람은 가치가 있다                 |
| 定義 / 正意  | 정의(定義)    | 정의(正意)             | 定義는 개념 정의, 正意는 바른 뜻        |
| 拒 / 距      | 막다(拒)      | 거리(距離)             | 拒는 손, 距는 발 관련 의미              |
| 占 / 店      | 점치다(占)    | 가게(店)               | 宀(집)이 있으면 店                      |
| 摩 / 魔 / 磨 | 문지르다(摩)  | 마귀(魔), 닦다(磨)     | 摩는 손, 魔는 귀신, 磨는 돌 석          |
| 兼 / 廉      | 겸하다(兼)    | 청렴(廉)               | 兼은 팔, 廉은 집 포함                   |
| 逐 / 遂      | 쫓다(逐)      | 이루다(遂)             | 逐은 선, 遂는 지붕(宀)                  |
| 徵 / 微      | 부르다(徵)    | 미세(微細)             | 徵은 王 포함, 微는 작음                 |
| 派 / 波      | 파벌(派系)    | 파도(波濤)             | 둘 다 물 수 변, 의미 다름               |
| 畫 / 書      | 그림(畵)      | 글(書)                 | 畵 아래에 밭 전(田) 포함                |
