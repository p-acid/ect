## 같은 부수 한자

---

### 寺(절 사)

| 한자 | 음  | 뜻        | 구성 성분       | 설명                        |
| ---- | --- | --------- | --------------- | --------------------------- |
| 寺   | 사  | 절, 절 사 | 자체            | 절이나 관청 등을 의미       |
| 時   | 시  | 시간, 때  | 日(해) + 寺(절) | 해가 절에 있음 → 때, 시간   |
| 詩   | 시  | 시(시문)  | 言(말) + 寺(절) | 절에서 말함 → 시문, 문학    |
| 持   | 지  | 가지다    | 扌(손) + 寺(절) | 손으로 절처럼 공손히 가지다 |
| 特   | 특  | 특별하다  | 牛(소) + 寺(절) | 절에 바치는 특별한 소       |

<br />

### 系(이을 계)

| 한자 | 음  | 뜻            | 구성 성분                                 | 설명                        |
| ---- | --- | ------------- | ----------------------------------------- | --------------------------- |
| 系   | 계  | 이을 계, 계통 | 자체                                      | 줄처럼 이어진 모양에서 유래 |
| 係   | 계  | 관계, 관련    | 亻(사람) + 系(계)                         | 사람 간의 관계를 이음       |
| 契   | 계  | 계약, 맺다    | 大(큰) + 刀(칼) + 卩(무릎 마디) + 系 변형 | 줄을 묶어 맺다 → 계약       |

<br />

### 黃(누를 황)

| 한자 | 음  | 뜻               | 구성 성분            | 관련 설명                             |
| ---- | --- | ---------------- | -------------------- | ------------------------------------- |
| 黃   | 황  | 누르다, 누런색   | 自(코) + 田(밭)      | 본래 ‘누런 빛깔’을 의미하는 기본 글자 |
| 廣   | 광  | 넓다, 넓은 집    | 广(집 엄) + 黃(황)   | 큰 지붕 아래 누런 땅 → 넓은 공간      |
| 擴   | 확  | 넓히다, 확장하다 | 扌(손) + 廣(넓을 광) | 손으로 넓힘 → 확장하다                |

<br />

### 糸/絲(실 사)

| 한자 | 음  | 뜻              | 구성 성분                        | 설명                                 |
| ---- | --- | --------------- | -------------------------------- | ------------------------------------ |
| 絲   | 사  | 실, 가는 실     | 자체                             | 실을 두 타래 겹쳐 놓은 모양          |
| 線   | 선  | 선, 줄          | 糸(실) + 泉(샘)                  | 물처럼 이어지는 실 → 선, 라인        |
| 緣   | 연  | 인연, 가장자리  | 糸(실) + 彖(단편)                | 실처럼 이어진 관계 → 인연            |
| 經   | 경  | 경전, 지나가다  | 糸(실) + 巠(지날 경)             | 실처럼 이어진 글 → 경전, 혹은 줄기   |
| 結   | 결  | 맺다, 묶다      | 糸(실) + 吉(길할 길)             | 실을 묶어 맺음                       |
| 編   | 편  | 엮다, 편집      | 糸(실) + 扁(작은 문)             | 실로 엮어 책을 만듦                  |
| 織   | 직  | 짜다, 직조      | 糸(실) + 戠(소리 직)             | 실을 이용해 천을 짬                  |
| 綿   | 면  | 솜, 부드러운 실 | 糸(실) + 帛(비단 백)             | 실을 풀어 만든 솜                    |
| 繼   | 계  | 계속하다        | 糸(실) + 絲(실)                  | 실로 이어 유지하다                   |
| 斷   | 단  | 끊다            | 糸(실) + 刀(칼 도) + 斤(도끼 근) | 실을 자른다.                         |
| 斷   | 연  | 잇닿다, 잇다    | 耳(귀 이) + 絲(실)               | 실로 귀를 잇다. 잇닿을 연(連)과 비교 |

<br />

### 曾(일찍이 증)

| 한자 | 음  | 뜻               | 구성 성분     | 설명                            |
| ---- | --- | ---------------- | ------------- | ------------------------------- |
| 曾   | 증  | 일찍이, 이전에   | 자체          | 본래 '층층이 쌓임', '이전'의 뜻 |
| 增   | 증  | 더하다, 증가하다 | 土(흙) + 曾   | 쌓인 흙 → 더해짐, 증가          |
| 憎   | 증  | 미워하다         | 忄(마음) + 曾 | 마음속에 쌓임 → 증오, 미움      |
| 贈   | 증  | 주다, 선물하다   | 貝(돈) + 曾   | 쌓인 재물을 줌 → 증여, 증정     |
| 僧   | 승  | 중               | 亻(사람) + 曾 | 절에 있는 중을 의미             |

<br />

### 走(달릴 주)

| 한자 | 음  | 뜻             | 구성 성분       | 설명                                |
| ---- | --- | -------------- | --------------- | ----------------------------------- |
| 走   | 주  | 달리다         | 자체            | 발이 땅에서 떨어진 사람 모양 → 달림 |
| 起   | 기  | 일어나다       | 走 + 己(자기)   | 자신이 달리며 일어남 → 시작, 기상   |
| 越   | 월  | 넘다           | 走 + 戉(도끼)   | 달려서 장애를 넘음 → 초월, 월경     |
| 趣   | 취  | 달려가다, 흥미 | 走 + 取(취하다) | 목적을 향해 달림 → 흥미, 의도       |
| 赴   | 부  | 가다, 나아가다 | 走 + 卜(점)     | 점친 방향으로 달려감 → 출발, 파견   |

### 扌(손 수)

| 한자 | 음  | 뜻         | 구성 성분          | 설명                       |
| ---- | --- | ---------- | ------------------ | -------------------------- |
| 投   | 투  | 던지다     | 扌 + 殳(몽둥이 수) | 손으로 몽둥이를 던짐       |
| 提   | 제  | 들다, 끌다 | 扌 + 是(옳을 시)   | 손으로 옳게 들다, 제시하다 |
| 拒   | 거  | 막다       | 扌 + 巨(클 거)     | 손으로 크게 막음 → 거절    |
| 持   | 지  | 가지다     | 扌 + 寺(절 사)     | 손으로 공손히 가짐         |
| 推   | 추  | 밀다       | 扌 + 隹(새 추)     | 손으로 새처럼 밀다 → 추진  |

### 隹(새 추)

| 한자 | 음  | 구성                           | 의미      | 설명                                                          |
| ---- | --- | ------------------------------ | --------- | ------------------------------------------------------------- |
| 隹   | 추  | 자체                           | 새 추     | 단독 사용은 드물지만, 다른 한자의 구성 요소로 자주 사용됨     |
| 集   | 집  | 隹(새 추) + 木(나무 목)        | 모을 집   | 새들이 나무에 모여 있는 모습에서 ‘모이다’는 의미 유래         |
| 進   | 진  | 隹(새 추) + 辶(쉬엄쉬엄 갈 착) | 나아갈 진 | 새가 앞으로 나아가는 모습에서 ‘전진하다’는 의미               |
| 雙   | 쌍  | 隹 + 隹 + 又(또 우)            | 쌍 쌍     | 두 마리 새를 손으로 잡고 있는 형상에서 ‘쌍’, ‘둘’의 의미 유래 |
