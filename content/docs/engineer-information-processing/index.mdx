## 필기시험

### 개요

- 시험 시간은 총 100분(1시간 40분)이며, 객관식 100문항으로 구성되어 있습니다.
- 과목별 배점은 20점이며 과목별 40점 이상, 전체 60점 이상을 취득해야 합격합니다.
- 시험 과목은 5과목으로 구성되어 있으며, 각 과목별로 20문항씩 출제됩니다.

### 과목 구성

| 과목      | 주요 내용            |
| --------- | -------------------- |
| **1과목** | 소프트웨어 설계      |
| **2과목** | 소프트웨어 개발      |
| **3과목** | 데이터베이스 구축    |
| **4과목** | 프로그래밍 언어 활용 |
| **5과목** | 정보시스템 구축 관리 |

### 학습 전략

- 시험 출제 방식이 **문제은행 방식**이기 때문에 최근 5년치 기출문제를 반복 풀이하는 형태로 학습합니다.
- 기출문제를 풀면서 부족한 부분을 보완하고, 이해가 안 되는 부분은 관련 자료를 찾아보며 학습합니다.
