## 프로젝트 계획 요약

<Callout type="warn">
  현재는 개인적인 이유로 **한자 암기**를 위한 프로젝트 형태로 운영 중입니다.
  향후엔 **영단어 암기 서비스** 형태로 전환할 예정입니다.
</Callout>

해당 프로젝트의 주요 목적은 <strong>효과적인 단어 암기</strong>입니다. 이를 위해 다음과 같은 기능을 구현할 예정입니다.

- **퀴즈 기능**: 지루한 단순 암기 방식에서 벗어나, <strong>게임화된 암기</strong>를 통해 학습의 재미를 더하고자 합니다.
- **오답노트 기능**: 간편하게 퀴즈에서 틀린 문제를 오답노트로 옮기고, **반복 학습**을 통해 기억을 강화할 수 있도록 합니다.
- **단어장 기능**: 사용자가 직접 단어를 추가하고 관리할 수 있는 기능을 제공합니다.
- **틀린 단어 분석 기능**: 사용자가 틀린 단어를 분석하여, 어떤 단어에서 자주 실수하는지를 파악할 수 있도록 합니다.

<br />

## 진행되지 않은 기능

- 인증 기능
- 오답노트 기능
- 단어장 기능
- 틀린 단어 분석 기능
